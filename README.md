## Install the repo


### Prerequisites

- [Poetry](https://python-poetry.org/docs/#installing-with-the-official-installer)


### Install

Clone repo, cd into it.

```sh
poetry install
```

## Download data

```sh
poetry run python data_collection/get_wikipedia_dataset.py
```
